# Excel-Bike-Sales-Dashboard
Bike Sales Dashboard (Excel)

# Project Overview

This project analyzes a bike sales dataset using Microsoft Excel to identify customer purchasing trends and create an interactive dashboard. The data was cleaned, transformed, and visualized to provide meaningful business insights.

This project was completed as part of Alex The Analyst's Data Analyst Bootcamp.

# Skills Demonstrated

- Data Cleaning
- Data Transformation
- Pivot Tables
- Pivot Charts
- Interactive Dashboards
- Slicers
- Data Visualization
- Excel Functions

# Tools Used

- Microsoft Excel

# Project Process

The following steps were completed:

- Cleaned and organized the raw dataset.
- Removed inconsistencies and prepared the data for analysis.
- Created additional columns to improve reporting.
- Built Pivot Tables to summarize the data.
- Designed Pivot Charts to visualize trends.
- Added Slicers to make the dashboard interactive.
- Created a professional dashboard to present key insights.

# Dashboard Insights

The dashboard allows users to analyze bike purchases based on:

- Marital Status
- Gender
- Age Group
- Income
- Education
- Region
